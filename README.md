

## Project Description

**Ab's Bite Menu** is a simple HTML webpage that displays the menu of a fictional restaurant.
The page is structured using basic HTML elements such as headings, paragraphs, lists, and links to organize the restaurant's offerings in a clear and readable format.

This project is designed as a beginner-friendly exercise to practice fundamental web development concepts.

---

## Features

The webpage includes the following sections:

* **Restaurant Title**
* **Introduction paragraph**
* **Menu Sections**

  * Appetizers
  * Main Courses
  * Desserts
* **Menu Items**

  * Each section contains at least three items
  * Each item includes a short description
* **Navigation Link**

  * A link at the bottom of the page that leads to a contact page placeholder.

---

## Technologies Used

* **HTML5**

This project uses only HTML to demonstrate basic webpage structure.

## How to Run the Project

1. Clone the repository from GitHub:

```
git clone https://github.com/your-username/restaurant-menu.git
```

2. Navigate into the project folder:

```
cd restaurant-menu
```

3. Open the `index.html` file in any web browser.

You can also open it using **VS Code** or any other code editor.
